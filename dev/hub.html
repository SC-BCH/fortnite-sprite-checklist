新しい JSON 駆動ルートの入口です。既存の dev/index.html は残したまま、viewer / editor をここから使えます。

JSON Viewer

draft JSON を読み込んで、チェック・保存画像・表示位置を確認する開発用 viewer です。画像はローカル選択で扱います。

viewer を開く

JSON Editor

項目を選んで画像上をクリックし、x / y / w / h を微調整して draft JSON を更新するための editor です。

editor を開く

Verification

editor と viewer の確認項目を固定したページです。publish へ上げる前の最終確認に使います。

verification を開く

Publish Guide

draft から publish へ昇格させるための manifest 形式と、version 付きファイル運用の考え方を固定したページです。

publish guide を開く

Legacy DEV

いままでの埋め込み画像ベースの開発ページです。新ルートとの比較用にそのまま残しています。

legacy を開く

使い方

  1. editor で対象画像を選択し、項目位置を調整します。
  2. editor 下部の JSON をコピーまたは保存します。
  3. 必要なら dev/boards/sprite-seirei.draft.json に反映します。
  4. viewer で同じ画像を選び、チェック位置・表示・PNG 保存を確認します。
  5. verification で確認項目を順にチェックします。
  6. publish guide で version 付き本番ファイル名と manifest 形式を確認します。
  7. 問題なければ、この JSON を publish 用 board ファイルへ昇格させます。
