`dev/index.html` は軽量な入口ページへ切り替えました。JSON 駆動の editor / viewer / verification / publish guide はここから辿れます。

Hub

dev 側の全ルートをまとめた入口です。迷ったらまずここを開きます。

hub を開く

JSON Viewer

draft JSON と選択画像を使って、チェック位置・表示・PNG 保存を確認します。

viewer を開く

JSON Editor

画像上で座標をクリック配置し、ドラッグ移動やリサイズで調整します。

editor を開く

Verification

publish 前に editor / viewer の確認項目を固定手順でチェックします。

verification を開く

Publish Guide

draft から publish へ昇格する時の manifest 形式と確認記録を参照します。

publish guide を開く

旧来の埋め込み画像型 `dev/index.html` はファイルが大きく、差し替えや最小変更でも壊しやすい構成でした。

今後の dev 運用は hub → editor → viewer → verification → publish guide を基本ルートにします。
